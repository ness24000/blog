# Newsletter plan 

Users should be able to subscribe to the blog in the following manner: 
1. Add their email to the newsletter. 
2. From then on, receive an email automatically whenever there is a new post. 
3. Be able to unsubscribe at any time by clicking on a link at the bottom of the emails they receive. 

In greater detail, each of these steps require: 

0. In general:
 - [x] A table to store email addresses. 
 - [x] An SMTP server: Mailtrap.

1. Subscription:
 - [ ] A page where they submit their email. 
 - [ ] A confirmation email.
